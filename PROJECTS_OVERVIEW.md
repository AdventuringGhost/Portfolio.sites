# AdventuringGhost Projects Overview

A comprehensive monorepo showcasing various web development, cloud infrastructure, and security projects.

## 🚀 Applications

### Portfolio (`apps/portfolio/`)
A modern, responsive portfolio website showcasing projects and skills with integrated notes system.
- **Status**: ✅ Complete
- **Tech Stack**: React, TypeScript, Vite, Tailwind CSS
- **Features**: Project showcase, notes system, responsive design

### Glow Grove (`apps/glow-grove/`)
A modern e-commerce demonstration application showcasing product catalogs, shopping cart functionality, and checkout processes.
- **Status**: 🚧 In Development
- **Tech Stack**: React, TypeScript, Docker
- **Features**: Product catalog, shopping cart, checkout process

### Hike Harvest Serverless (`apps/hike-harvest-serverless/`)
A serverless API for managing hiking recipes and adventure data using AWS Lambda and DynamoDB.
- **Status**: 🚧 In Development
- **Tech Stack**: AWS Lambda, DynamoDB, Serverless Framework
- **Features**: Recipe management, serverless architecture

## 🔧 Backend Services

### Blogsite API (`apps/blogsite-api/`)
A RESTful API service for managing blog posts, comments, and user authentication.
- **Status**: 🚧 In Development
- **Tech Stack**: Node.js, Express, MongoDB
- **Features**: User management, content management, security

## ☁️ Cloud & Infrastructure

### NetOps Showcase AWS (`apps/netops-showcase-aws/`)
A comprehensive cloud networking project demonstrating VPC design, security groups, and AWS networking best practices.
- **Status**: 🚧 In Development
- **Tech Stack**: AWS VPC, Security Groups, Route Tables
- **Features**: Multi-AZ deployment, security best practices

### Security Lab (`apps/security-lab/`)
A comprehensive security laboratory demonstrating IAM policies, WAF configurations, and AWS security best practices.
- **Status**: 🚧 In Development
- **Tech Stack**: AWS IAM, WAF, CloudFormation
- **Features**: Security policies, access controls, monitoring

## 🏗️ Infrastructure as Code

### Terraform (`terraform/`)
Infrastructure as Code using Terraform for managing cloud resources and deployments.
- **Status**: 🚧 In Development
- **Tech Stack**: Terraform, AWS Provider
- **Features**: Resource management, state management

### AWS CDK (`cdk/`)
Infrastructure as Code using AWS Cloud Development Kit (CDK) for TypeScript-based cloud resource management.
- **Status**: 🚧 In Development
- **Tech Stack**: AWS CDK, TypeScript
- **Features**: Type-safe infrastructure, deployment pipelines

## 📦 Shared Packages

### UI Package (`packages/ui/`)
Shared UI components and design system.
- **Status**: ✅ Complete
- **Tech Stack**: React, TypeScript, Tailwind CSS
- **Features**: Reusable components, consistent styling

### Types Package (`packages/types/`)
Shared TypeScript type definitions across projects.
- **Status**: ✅ Complete
- **Tech Stack**: TypeScript
- **Features**: Type safety, shared interfaces

## 🎯 Project Goals

- **Learning**: Demonstrate various technologies and best practices
- **Portfolio**: Showcase development skills and project experience
- **Infrastructure**: Practice cloud architecture and DevOps principles
- **Security**: Implement security best practices and compliance

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Choose a project and follow its specific README
